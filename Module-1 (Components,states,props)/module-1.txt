Q-5 =>  What is Components in React Js? 
A-5 => A Component is one of the core building blocks of React. In other words, we 	can say that every application you will develop in React will be made up of 	pieces called components.
	Two type of compnent:-
	1=> Class Components:- The class components are a little more complex than the 	functional components.
	Ex.
	class Democomponent extends React.Component
	{
    	render(){
          return <h1>Welcome Message!</h1>;
    		}
	}
	
	2=> Functional Components:- Functional components are simply javascript 	functions. We can create a functional component in React by writing a 	javascript function.
	Ex.
	const Democomponent=()=>
	{
    		return <h1>Welcome Message!</h1>;
	}
